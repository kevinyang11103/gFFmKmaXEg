# 金融管理系统 python675

## 项目概述

金融管理系统，又称个人财务管理系统，采用 Bootstrap + Django 技术栈开发。本项目旨在为个人用户提供便捷的财务管理和分析工具。

## 技术栈

- 前端：Bootstrap
- 后端：Django

## 功能模块

1. 用户管理
   - 用户注册与登录
   - 用户信息管理
2. 财务管理
   - 收入与支出记录
   - 预算设置与监控
   - 财务报表生成
3. 账户管理
   - 银行卡与第三方支付账户管理
   - 账户余额与交易记录查询

## 系统角色

- 普通用户：可注册、登录系统，管理个人财务信息
- 管理员：负责系统运维与用户管理

## 运行环境

- Python 3.6 或更高版本
- Django 2.2 或更高版本
- Bootstrap 4.x

## 部署步骤

1. 克隆项目代码到本地
2. 安装依赖库：`pip install -r requirements.txt`
3. 配置数据库（如使用 MySQL 或 PostgreSQL）
4. 运行迁移：`python manage.py migrate`
5. 启动开发服务器：`python manage.py runserver`

## 目录结构

```
finance_system/
├── apps/
│   ├── user/  # 用户管理模块
│   └── finance/  # 财务管理模块
├── static/  # 静态文件目录
├── templates/  # HTML模板目录
├── manage.py
└── requirements.txt  # 依赖文件
```
